🏦 Bank App — Система управления банковскими вкладами
Веб-приложение для управления личными финансами и банковскими продуктами. Реализовано на Go с использованием архитектуры, ориентированной на безопасность, масштабируемость и удобство поддержки.
🚀 Особенности проекта
Безопасность: Хеширование паролей (bcrypt), защищенные зашифрованные сессии (gorilla/sessions).
Архитектура: Четкое разделение на слои (Handlers, Service, Repository).
Конфигурация: Централизованное управление настройками через YAML и переменные окружения.
База данных: Автоматическое применение миграций при запуске, использование транзакций для финансовых операций.
Логирование: Структурированные цветные логи с уровнями важности (Zap).
Docker: Полная контейнеризация приложения и базы данных для быстрого развертывания.
