# 🏦 Bank App

**Bank App** — веб-приложение для управления личными финансами и банковскими вкладами, разработанное на **Go (Golang)**.

Проект реализован с использованием многослойной архитектуры, ориентированной на **безопасность**, **масштабируемость** и **простоту сопровождения**.

# 🚀 Особенности проекта

- 🔐 **Безопасность**
  - Хеширование паролей с использованием **bcrypt**
  - Защищённые зашифрованные сессии через **gorilla/sessions**

- 🏗 **Чистая архитектура**
  - Разделение на слои:
    - Handlers
    - Service
    - Repository

- ⚙ **Конфигурация**
  - Централизованная настройка через YAML и переменные окружения

- 🗄 **Работа с базой данных**
  - PostgreSQL
  - Автоматическое применение миграций при запуске
  - Использование транзакций для финансовых операций

- 📄 **Логирование**
  - Структурированные цветные логи
  - Различные уровни логирования
  - Реализовано на **Uber Zap**

- 🐳 **Docker**
  - Полная контейнеризация приложения
  - Docker Compose для быстрого запуска

# ⚡ Быстрый старт

## Способ 1. Docker (Рекомендуется)

Приложение автоматически:

- создаст контейнер PostgreSQL;
- применит все миграции;
- запустит веб-сервер.

```bash
docker-compose up --build
```

После запуска приложение будет доступно по адресу:

```
http://localhost:8080
```

---

## Способ 2. Локальный запуск

### 1. Установите зависимости

- Go 1.21+
- PostgreSQL 15+

### 2. Создайте базу данных

```sql
CREATE DATABASE bank_deposits;
```

### 3. Создайте файл `.env`

Пример:

```env
DB_USER=postgres
DB_PASS=your_password
DB_NAME=bank_deposits
DB_HOST=localhost
SESSION_SECRET=super-secret-key-for-production
```

> При запуске через Docker вместо `localhost` используйте `db`.

### 4. Запустите приложение

```bash
go run main.go
```

## 🏦 Банковские вклады

Поддерживается **5 различных типов вкладов**:

- Гибкий
- Накопительный
- Срочный
- Премиум
- Специальный

Для каждого вклада доступны:

- открытие;
- пополнение;
- частичное снятие средств;
- начисление процентов;
- закрытие.

---
